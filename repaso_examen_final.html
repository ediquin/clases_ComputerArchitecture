🔧 Unidad Central de Procesamiento (CPU)

La CPU es el "cerebro" de la computadora, responsable de ejecutar instrucciones y coordinar todas las operaciones del sistema.

Características Principales:

  • Procesamiento Serial: Ejecuta instrucciones una tras otra de forma secuencial
  • Núcleos Complejos: Pocos núcleos (4-32) pero muy potentes y versátiles
  • Alta Velocidad de Reloj: Frecuencias muy altas (2-5 GHz)
  • Versatilidad: Puede manejar cualquier tipo de tarea computacional
  • Predicción de Saltos: Tecnologías avanzadas para optimizar ejecución

💡 Concepto Fundamental

Las CPUs están optimizadas para velocidad y versatilidad en tareas secuenciales complejas. Su arquitectura prioriza la ejecución rápida de instrucciones individuales.

🎮 Unidad de Procesamiento Gráfico (GPU)

Las GPUs fueron diseñadas originalmente para procesar gráficos, pero han evolucionado hacia la computación de propósito general.

Características Principales:

  • Procesamiento Paralelo: Miles de núcleos trabajando simultáneamente
  • Núcleos Simples: Muchos núcleos (1000+) especializados en operaciones básicas
  • Operaciones Repetitivas: Ideal para tareas que requieren el mismo cálculo múltiples veces
  • Alto Throughput: Procesa grandes volúmenes de datos simultáneamente
  • Arquitectura SIMD: Una instrucción, múltiples datos

💡 Concepto Fundamental

Las GPUs están optimizadas para paralelismo masivo. Sacrifican versatilidad por la capacidad de procesar miles de operaciones similares simultáneamente.

📡 Comunicación Paralela vs Serial

Los sistemas de comunicación pueden transmitir datos de diferentes maneras, cada una con sus ventajas y limitaciones.

Comunicación Paralela:

  • Múltiples Canales: Varios bits se transmiten simultáneamente
  • Mayor Ancho de Banda: Más datos por unidad de tiempo
  • Distancias Cortas: Efectiva en rangos limitados
  • Problemas de Sincronización: Dificultad para mantener sincronía en largas distancias

Comunicación Serial:

  • Un Canal: Los bits se transmiten uno tras otro
  • Largas Distancias: Mejor para transmisión a larga distancia
  • Menor Complejidad: Más simple de implementar y mantener

⚠️ Importante

La comunicación paralela NO fue desarrollada para largas distancias. Los problemas de sincronización y interferencia la hacen más efectiva en distancias cortas.

Las GPUs tienen más núcleos que las CPUs, pero son núcleos más simples. Es un trade-off entre cantidad y complejidad.

📊 Cálculos de Tiempo de Procesamiento

Para entender el rendimiento de un procesador, es fundamental calcular el tiempo que tarda en ejecutar instrucciones individuales.

🧮 Fórmulas Fundamentales

Período (T) = 1 / Frecuencia (f)

Donde:

  • T = Tiempo de un ciclo de reloj (segundos)
  • f = Frecuencia del procesador (Hz)
  • 1 Hz = 1 ciclo por segundo
  • 1 GHz = 1,000,000,000 Hz = 10⁹ Hz

📝 Metodología de Cálculo

Paso 1: Identificar Datos

Extraer la frecuencia del procesador y convertir a Hz si es necesario

Paso 2: Aplicar Fórmula

Usar T = 1/f para calcular el período

Paso 3: Realizar Cálculo

Efectuar la división y expresar en notación científica

Paso 4: Interpretar Resultado

El resultado estará en segundos, típicamente nanosegundos para procesadores modernos

💡 Ejemplo Conceptual

Un procesador de alta frecuencia puede ejecutar miles de millones de ciclos por segundo. Cada ciclo representa una oportunidad para ejecutar una instrucción básica, lo que explica la alta velocidad de procesamiento de las computadoras modernas.

🔑 Conceptos Clave

  • Mayor frecuencia = menor tiempo por instrucción
  • Los tiempos se miden en nanosegundos (10⁻⁹ segundos)
  • Una instrucción básica = un ciclo de reloj
  • La frecuencia determina la velocidad máxima de procesamiento
