Conceptos básicos de hardware y software en sistemas computacionales

Clase 1 – Arquitectura de Computadoras

1. Introducción

¿Qué es un sistema computacional?

Un sistema computacional es un conjunto de hardware y software que trabajan juntos para recibir, procesar, almacenar y transmitir información.

Ejemplo:

  • Hardware: Pantalla, batería, procesador, cámara.
  • Software: Sistema operativo (Android, iOS), aplicaciones (WhatsApp, navegador, calculadora).

2. Explicación Teórica

A. Hardware: El lado físico de la computadora

  • Unidad Central de Procesamiento (CPU): Realiza operaciones matemáticas y lógicas. Ejemplos: Procesadores Intel, AMD.
  • Memoria y almacenamiento:
    • RAM: Memoria temporal.
    • ROM: Contiene información permanente.
    • Almacenamiento secundario: HDD, SSD.
  • Dispositivos de entrada y salida: Teclado, mouse, monitor, impresora.
  • Placa madre y buses: Conecta los componentes y permite la comunicación.

B. Software: El cerebro del sistema

  • Sistema Operativo: Administra recursos y permite la interacción. Ejemplos: Windows, Linux, macOS.
  • Software de aplicación: Programas para tareas específicas como Word, Photoshop.
  • Software de desarrollo: Herramientas para programar, como Python, Visual Studio.
